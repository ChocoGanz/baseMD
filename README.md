# Base Bot Multi Device
```Note : Script ini hanya berisi mentahan base multi device```
# Penginstalan untuk termux
- pkg update && pkg upgrade
- pkg install nodejs
- pkg install git
- git clone https://github.com/ChocoGanz/baseMD
- cd chocoMD
- npm install
- node index.js
# Jika penginstalan error
- npm i -g npm@6
- npm install
- node index.js
